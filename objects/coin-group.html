Coin Group

Coin Groups are objects which automatically fill their volume with Coins, its recomended that you use coin groups instead of regular coins for those reasons:

  1. Performance
    • Regular coins do a distance check every frame in order to occlude themselfes if Mart is too far away, therefore, using too much regular coins can decrease performance. Coin Groups in the other hand, do one distance check for all coins in that group.
  2. Ease of use
    • Using Coin Groups is easier than using regular coins, since by just adjusting the size of the Coin Group new coins get created, it saves time that would've been spent CTRL + D regular coins.
  3. Level data size
    • When using regular coins, each coin is an object, and the more objects you use, the closer you get to reaching the level data size limit. With Coin Groups, all coins in a Coin Group count as just one object.
