Manic Mania Stud (collide)

This object inherits all the properties from the Stud object, but this one can only collide with Mart during Manic Mania.

Properties

Size

Determines this object's size

Positioning

This property can only be edited with the position/rotation handle. The value corresponds to where the Manic Mania Stud (collide) is in the level.

Color

Determines this object's color.

Material

Determines this object's appearance

Transparency

Determines how transparent the object is, this value ranges from 0 to 1. 1 is fully transparent while 0 is fully opaque

Reflectance

An property which defines how reflective this object is. 1 is totally reflective, and 0 doesn't reflect anything at all.

Shape

Determines this object's shape. Those shapes are:

  1. Ball
    • A spherical shape.
  2. Block
    • A block shape.
  3. Cylinder
    • A cylinder shape.
  4. Wedge
    • A wedge shape with a slope on one side.
  5. Corner Wedge
    • A wedge shape with slopes on two sides.

Linear velocity

A Vector3 value that determies the object's linear velocity.

When either the X, Y or Z values are not 0, Mart's velocity will be affected by the object's velocity.

Angular Velocity

A Vector3 value that determies the object's angular velocity

This one is a little more complicated, and there aren't visual examples of it in action! (yet)

On top of that, Mart can't interact with Angular Velocity, only unanchored objects!
